# 책 빌드 및 미리보기

GitBook 프로젝트를 빌드하고 검증합니다.

## 사용법
```
/build-book
```

## 실행 절차

### 1단계: 구조 검증
- SUMMARY.md의 모든 링크가 실제 파일과 매칭되는지 확인
- 누락된 파일이나 잘못된 경로 보고

### 2단계: 콘텐츠 검증
- 각 마크다운 파일의 H1 제목 존재 확인
- 이미지 참조의 유효성 확인
- 코드 블록 언어 태그 확인

### 3단계: 빌드
- `npx gitbook-cli build` 또는 로컬 빌드 도구 실행
- 빌드 에러 보고 및 수정 제안

### 4단계: 결과 보고
- 총 챕터/섹션 수
- 발견된 이슈 목록
- 빌드 성공/실패 여부
