# 말동무에서 동료로

AI도 아이처럼 자라요. 처음에는 말도 제대로 못 하던 아기였지만, 지금은 혼자서 마트도 다녀오는 어른이 됐어요.

이 장에서는 AI가 어떻게 네 단계를 거쳐 성장해왔는지를 살펴볼 거예요. 어렵게 느껴지실 수 있지만, 걱정 마세요. 아이가 태어나서 자라는 이야기라고 생각하시면 돼요.

## 이 장에서 배우는 것

- **1세대**: 글자로만 대화하던 시절 — AI가 갓 태어난 아기였을 때
- **2세대**: 사진과 음성을 이해하기 시작한 시절 — 눈과 귀가 생긴 아이
- **3세대**: 인터넷을 검색하고 문서를 읽는 시절 — 도구를 쓰는 학생
- **4세대**: 목표만 주면 알아서 해내는 지금 — 스스로 일하는 어른

## 왜 이걸 알아야 할까요?

AI의 성장 과정을 알면, 지금 AI가 무엇을 할 수 있고 무엇을 못 하는지 자연스럽게 이해하게 돼요. 마치 다섯 살 아이한테 운전을 시키지 않듯이, AI의 현재 수준을 알면 적절한 기대와 활용이 가능해져요.

자, 그럼 AI의 성장 앨범을 함께 넘겨볼까요?
