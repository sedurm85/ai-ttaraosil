# AI에게 말 거는 법

AI를 처음 만나면 뭐라고 말해야 할지 막막하죠? 컴퓨터한테 말을 건다는 게 어색하기도 하고요. 하지만 걱정 마세요. AI에게 말 거는 건 어려운 프로그래밍이 아니에요. 그냥 친구한테 이야기하듯이 하면 돼요.

이 장에서는 AI와 자연스럽게 대화하는 방법을 배워볼 거예요. "명령"이 아니라 "대화"라는 게 핵심이에요. 식당에서 "밥 줘!"라고 하는 것과 "오늘 좀 피곤한데, 따뜻하고 든든한 거 뭐가 있을까요?"라고 하는 것은 전혀 다른 결과를 가져오잖아요. AI도 마찬가지예요.

## 이 장에서 배우는 것

- **명령이 아닌 대화**: AI에게 딱딱하게 지시하는 게 아니라, 내 상황을 설명하며 자연스럽게 대화하는 법을 배워요.
- **좋은 질문 만드는 법**: 같은 질문이라도 어떻게 물어보느냐에 따라 AI의 답이 완전히 달라져요. 좋은 질문의 비결을 알려드릴게요.
