# 일상에서 AI 활용하기

AI 사용법을 배웠으니, 이제 실제 생활에서 써볼 차례예요! 이 장에서는 우리가 매일 겪는 일상 속에서 AI를 어떻게 활용할 수 있는지 구체적인 예시를 모아봤어요.

요리, 여행, 글쓰기처럼 누구나 하는 일에 AI를 살짝 끼워 넣으면, 생활이 놀라울 만큼 편해져요. 거창한 기술이 아니라, 생활의 작은 도우미로 AI를 만나보세요.

## 이 장에서 배우는 것

- **요리 도우미**: 냉장고 재료를 활용한 레시피 추천받는 법
- **여행 가이드**: 내 상황에 딱 맞는 여행 일정 만드는 법
- **글쓰기 도우미**: 공지문, 보고서, 경조사 문자까지 AI와 함께 쓰는 법
- **매일 5분 AI 습관**: 꾸준히 AI와 친해지는 방법
