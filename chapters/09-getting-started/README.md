# AI 도구 시작하기

드디어 직접 AI를 써볼 시간이에요! 지금까지 AI가 뭔지, 어떻게 대화하는지, 일상에서 어떻게 쓸 수 있는지 배웠죠? 이제 진짜로 내 스마트폰이나 컴퓨터에 AI를 설치하고, 첫 대화를 나눠볼 거예요.

걱정하지 마세요. 카카오톡 쓸 줄 알면 AI도 쓸 수 있어요. 정말이에요. AI에게 말을 거는 건 친구한테 카톡 보내는 것과 거의 똑같아요. 글자를 입력하고, 전송 버튼을 누르면 끝이에요.

이 장에서는 가장 많이 쓰이는 AI 도구 세 가지를 하나씩 설치하고 사용해볼 거예요. 그리고 스마트폰으로 할지, 컴퓨터로 할지 고민되는 분들을 위한 가이드도 준비했어요.

## 이 장에서 배우는 것

- **ChatGPT 시작하기**: 가장 유명한 AI, 설치부터 첫 대화까지
- **Claude 시작하기**: 꼼꼼한 설명이 필요할 때 좋은 AI
- **Gemini 시작하기**: 구글 계정만 있으면 바로 쓸 수 있는 AI
- **스마트폰 vs 컴퓨터**: 어떤 기기로 시작하면 좋을지 안내
