# 프롬프트 실전 연습

> AI 도구를 설치했으니, 이제 직접 대화해볼 시간이에요!

---

## 이 챕터에서 배울 것

앞에서 ChatGPT, Claude 같은 AI 도구를 설치하고 준비를 마쳤어요. 이제 진짜 중요한 단계예요 — **AI와 실제로 대화하는 연습**이에요.

처음에는 AI에게 뭐라고 말해야 할지 막막할 수 있어요. 마치 외국에 처음 가서 현지인에게 말을 걸 때처럼요. 하지만 걱정하지 마세요! 이 챕터에서 바로 써먹을 수 있는 대화법을 알려드릴게요.

### 학습 목표

- AI와의 첫 대화에 대한 두려움 없애기
- 바로 복사해서 쓸 수 있는 실전 프롬프트 템플릿 익히기
- 대화를 이어가며 더 좋은 답변 받는 방법 배우기
- AI가 이상한 답을 줄 때 대처하는 법 알기
