# AI로 만들어보기

지금까지 우리는 AI에게 **질문하는 법**을 배웠어요. 궁금한 것을 물어보고, 답을 얻고, 더 좋은 답을 받기 위해 질문을 다듬는 연습도 했죠.

이제 한 단계 더 나아갈 거예요. AI에게 질문만 하는 게 아니라, **AI와 함께 무언가를 만들어볼** 거예요!

## 이 챕터에서 배우는 것

AI를 활용해서 실생활에 바로 쓸 수 있는 것들을 직접 만들어볼 거예요.

- **이미지 만들기** — 그림을 못 그려도 AI가 대신 그려줘요
- **글 다듬기** — 공지문, 문자, 편지를 멋지게 완성해요
- **표와 목록 정리** — 복잡한 정보를 깔끔하게 정리해요
- **나만의 활용법 찾기** — 여러분의 생활에 맞는 AI 사용법을 발견해요
